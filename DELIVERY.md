# TikTok Bumper Car Video Delivery

This repository stores the delivery package for the bumper-car TikTok video project.

## Main outputs

- `tiktok-bumper-car-package.zip` contains the complete package.
- Final HeyGen enhanced video inside the zip: `assets/bumper-car-heygen-final-v2.mp4`
- Local source-footage edit inside the zip: `assets/bumper-car-tiktok-v2.mp4`
- Contact sheet preview: `assets/bumper-car-heygen-final-v2-contact.jpg`
- Production notes: `README-production.md`
